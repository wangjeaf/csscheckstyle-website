CKstyle 利用 Python 的 easy_install，一键安装与更新

详细的安装流程

  1. 下载并安装 ActivePython
  2. 运行 easy_install https://github.com/wangjeaf/CSSCheckStyle/archive/master.tar.gz 自动安装
  3. 再次运行即可更新

命令行工具

请移步 github ，或在安装后通过 ckstyle/fixstyle/csscompress --help 查看帮助~~

CKService 让您在不需要安装 Python 和 CKstyle的情况下，一键分析网站 CSS 的压缩情况和优化空间

针对本站的CKService结果

什么是CKService

目前，CKStyle已经做成了一个开放的服务CKService，通过访问服务的URL，并给出特定的参数，即可实时获悉CSS代码处理以后的结果

有了CKService，通过一次点击，即可知道：

  • 本网站的所有CSS能被压缩到多小，压缩百分比
  • 一千万次PV时能够节省多少 流量
  • 在页面上 实时替换和恢复 压缩后的CSS文件，查看压缩后对页面的影响（轻松无压力的试用吧！）

感谢您为CKstyle报bug

在使用CKstyle的Python版本，或者使用本官网时出现的任何问题，都可以在github上提出来。

当为Python版本报bug时，为了更好的了解CKstyle在您使用过程中遇到的问题，请您：

  1. 描述您电脑上的 Python 版本
  2. 给出您测试时使用的 CSS 源代码
  3. 给出您期望的 CKstyle 处理结果
  4. 给出您觉得有问题的 CKstyle 处理后的结果
  5. 其他您想说的~~~
