CKstyle 利用 Python 的 easy_install，一键安装，方便快捷。

详细的安装流程

  1. 下载并安装 ActivePython
  2. 执行 easy_install，确认 easy_install 命令存在
  3. 运行 easy_install https://github.com/wangjeaf/CSSCheckStyle/archive/master.tar.gz 自动安装
  4. 运行 fixstyle --help 确认安装是否成功
